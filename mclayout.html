FrenchGuy's Steamdeck Minecraft Layout

FrenchGuy's Steamdeck Minecraft Layout All keys are the keys in Minecraft QWERTY layout. This should work too if you use the Deck's AZERTY layout or others, but I don't guarentee it!
L1 - Mouse scroll up // Go left in hotbar
R1 - Mouse scroll down // Go right in hotbar
L2 - Place block/interact
R2 - Break block/attack
The weird small button on the left - Tab
The three bars hamburger menu on the right button - Escape key (basically pause and close UI)
A - Jump/Space
B - Escape too
X - L4J crafting menu // Open vanilla inventory
Y - L4J Inventory
Right stick click (R3) - Shift (Sneak)
Left stick click (L3) - Ctrl key (Sprint)
Left stick movement - Move character
Right trackpad movement/Right stick movement - Move camera
Right trackpad click - Same as R2
Directional pad
Up - Scroll wheel click (Pick block)
Down - Q key (Drop item)
Left - F key (Put item to second hand)
Right - T key (Open chat)
